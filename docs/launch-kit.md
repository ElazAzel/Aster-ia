## 90-second Demo Script

1. Open Asterion AI and start the FastAPI sidecar from the Desktop Sidecar panel.
2. Show Privacy Radar in green local mode.
3. Select `llama3.2` and send a streaming local chat message.
4. Switch to a Context Room and show room-scoped memory policy.
5. Show Knowledge Vault document metadata and run RAG search.
6. Export a Research Receipt into an artifact.
7. Generate an AgentPlan in Agent Lab.
8. Create an AgentRun and show Flight Recorder logs.
9. Close with: local-first data control, visible privacy, transparent agents.

## Hacker News Show HN Draft

```text
Show HN: Asterion AI — a local-first desktop AI Control Room

I am building Asterion AI, an open-source Tauri + FastAPI desktop workspace for local models, encrypted memory, document RAG, deep research, and transparent agent runs.

The focus is not another chat UI. The main idea is visible control:
- Privacy Radar shows whether a task stays local, becomes hybrid, or needs external consent.
- Context Rooms scope files, memory, model policy, and retention.
- Research Receipts tie claims back to sources.
- Agent Lab creates a plan first and records every action through a Flight Recorder.

Stack: Tauri v2, Svelte, FastAPI, Ollama, SQLite + SQLCipher, LanceDB, DuckDB, SearXNG, ComfyUI.
```
