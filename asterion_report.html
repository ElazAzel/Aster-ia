01

Суть проекта

Asterion AI — это open-source AI workspace нового поколения: персональная AI-операционная среда, которая объединяет локальные нейросети, внешние API, чат, агентов, документы, глубокие исследования, изображения, голос, автоматизации и личную базу знаний в одном интерфейсе.

«Не просто окно для общения с нейросетью — полноценная рабочая среда, где AI-ответы превращаются в полезные объекты.»

Центральная идея

Современные AI-продукты вынуждают пользователя вручную переносить результаты в другие приложения. Asterion строится на иной логике: пользователь ставит задачу, система доводит её до готового артефакта — документа, таблицы, отчёта, кода, изображения, workflow.

Главная формула проекта

AI workspace для людей, которым нужен не просто ответ, а управляемый рабочий результат.

Пять ключевых столпов

🔮
Adaptive Artifacts
Ответы превращаются в рабочие объекты: документы, таблицы, research-отчёты, код, изображения — всё редактируемо и экспортируемо.
🔒
Privacy Radar
Приватность не в README, а в интерфейсе. Пользователь видит в реальном времени, что обрабатывается локально, а что уходит наружу.
🗂
Context Awareness
Context Rooms, Knowledge Vault и Memory Ledger дают пользователю полный контроль над тем, что AI «знает» и «помнит».
🤖
Agent Transparency
Агенты работают через план (Task Simulator), разрешения и полный лог (Flight Recorder). Никаких чёрных ящиков.
🏗
Workspace Thinking
Чат, документы, research, images, vault и automations — единая система, а не набор несвязанных инструментов.

02

Рыночный контекст и конкуренты

В 2025–2026 годах рынок AI-инструментов пережил кризис доверия к облачным провайдерам. Ужесточение GDPR/HIPAA, корпоративные утечки и рост open-source LLM создали спрос на локальные и гибридные решения. По аналитике, к концу 2026 года до 40% корпоративных AI-приложений будут работать на собственной инфраструктуре компаний.

Главный конкурент: Odysseus

Самый близкий аналог по духу — Odysseus, self-hosted AI workspace от PewDiePie (запуск ~31 мая 2026, более 34K звёзд на GitHub за месяц). Python FastAPI + простой frontend, Docker. Включает чат, агентов, Cookbook (270+ моделей), Deep Research, документы, память, email/calendar.

Ключевое отличие Asterion от Odysseus

Odysseus — широкий, но приватность и безопасность агентов не являются первоклассными фичами интерфейса. Asterion делает Privacy Radar и Agent Transparency видимым слоем UI, а не пунктом в документации.

Лицензионный ландшафт (важно)

Open WebUI в апреле 2025 перешёл с BSD-3 на ограничительную лицензию — это создало спрос на MIT/Apache2 альтернативы. LM Studio сделал продукт бесплатным для работы, сместив монетизацию в Teams/Enterprise. Asterion с MIT-лицензией занимает это освободившееся пространство.

04

UX/UI концепция

Интерфейс Asterion AI — не GPT-клон. Он вдохновлён Claude (артефакты, спокойствие, структура), Linear (скорость, Cmd+K), Figma (canvas, панели, версии), Arc (spaces, боковая навигация) и Notion (блоки, документы). Метафора продукта — «AI Control Room».

Главное UX-правило

Пользователь работает не с бесконечной перепиской, а с живыми AI-артефактами. Каждый ответ — это объект, который можно редактировать, экспортировать, передать агенту.

Трёхколоночный layout

Основная структура: [Left Sidebar] [Main Workspace] [Context / Artifact Panel]

  • Left Sidebar — навигация: Home, Chat, Research, Documents, Vault, Agents, Images, Automations, Models, Settings
  • Main Workspace — основная рабочая область: чат, документ, research canvas, изображение, workflow
  • Context Panel (справа) — модель, файлы, источники, память, приватность, настройки текущего объекта
  • Status Bar (внизу) — Privacy status, активная модель, VRAM/RAM, соединение

Три UX-режима

Simple Mode
Большие кнопки, автовыбор модели, скрытые параметры inference. Privacy Radar — упрощённый: «Безопасно» / «Часть данных уйдёт». Для 80% пользователей.
Pro Mode
Полный контроль: MCP, shell, квантизация, температура, векторный индекс, API routing, полные логи. Для разработчиков и power users.
Safe / Airgap
Интерфейс «краснеет», все внешние элементы блокированы визуально. Принудительный офлайн. Для корпоративных данных.

Уникальные UX-паттерны

Privacy Radar (выдающийся UI-паттерн)

Перед отправкой запроса пользователь видит компактную полосу:

📄 2 PDF (локально) → 🧠 Claude API (внешний) → ⚠️ Данные уйдут в Anthropic
[ Запустить ] [ Только локально ] [ Убрать документы ] [ Изменить ]

Human Error Layer

Вместо технических ошибок — понятные карточки с вариантами решения:

⚠ Не хватило видеопамяти (VRAM)
Модель требует 40 GB VRAM, у вашей карты 8 GB.
Варианты: ⚡ Авто-квантизация Q4 · 🐢 CPU offload · ☁️ Запустить через API

Red Button Mode

Кнопка «Остановить всё» (Cmd+Esc): останавливает агентов, отключает внешние запросы, блокирует shell, сохраняет лог, предлагает очистить временные данные.

05

Технический стек

Консенсус всех источников: Tauri v2 + Python FastAPI + Ollama

Все 6 AI-систем независимо пришли к одному и тому же рекомендуемому стеку для MVP. Это сильный сигнал.

СлойТехнологияОбоснование
Desktop shellTauri v2 (Rust)96% меньше размер, ~50% меньше RAM vs Electron. Нативный UX. Строгая безопасность IPC. Высвобождает ресурсы для LLM.
FrontendSvelte 5 / ReactБыстрый рендеринг, реактивность. Svelte предпочтительнее для production desktop.
BackendPython FastAPIПроверено в Odysseus. Удобен для AI-задач. Опционально Rust sidecar для критичных частей.
МоделиOllama (MVP) + llama.cpp / vLLMOllama: простой reliable runtime для MVP. vLLM для power users.
Векторная БДLanceDB (embedded)Serverless, хранит векторы прямо на диске, минимум RAM. Qdrant / Chroma как альтернатива.
Основная БДSQLite + SQLCipherAES-256 шифрование on-the-fly, ключи через OS keychain. Embedded, никаких серверов.
АналитикаDuckDBКолоночная БД для Deep Research: быстрые запросы к большим CSV без загрузки в RAM.
Агенты / инструментыMCP (Model Context Protocol)Стандарт Anthropic. Совместимость с экосистемой Claude Desktop, LocalAI. Безопасные разрешения.
ИзображенияComfyUI / Diffusers (локально)Bridge к локальному SD/FLUX. Stable Diffusion XL Turbo для RTX 4060.
ПоискSearXNG (локальный инстанс)Приватный поиск без трекинга для Deep Research.
Голосwhisper.cpp / MoonshineЛокальная транскрибация без облака.
СекретыOS keychain (Keychain/Credential Mgr)API-ключи никогда не хранятся на диске в открытом виде.

Варианты развёртывания

Desktop App (Tauri)
Для обычных пользователей. Signed installer, auto-update, нативный UX. Приоритет для MVP.
Docker Compose
Для self-hosted / продвинутых пользователей и команд. Все сервисы (app, ollama, searxng) в одной команде.
Manual Dev Mode
Для разработчиков, contributing к проекту. Полный контроль над каждым компонентом.
Team Server
Docker/K8s для команд. Multi-user, роли, shared workspaces. Монетизация Phase 7.

Особенность: Multi-Window State Sync

В Tauri v2 каждое окно — изолированный Webview. Для синхронизации состояния между окнами (например, чат на одном мониторе, canvas на другом) используется Rust-опосредованный IPC-мост: глобальное состояние в Mutex<AppState>, широковещательные события, revision gates против устаревших пакетов и state coalescing при частых изменениях (60 FPS без задержек).

06

Бизнес-модель

Золотое правило монетизации

Нельзя ломать доверие. Если продукт обещает контроль и приватность — монетизация не должна превращать его в закрытый облачный сервис. Деньги берём только за удобство, масштаб и premium-ценность.

Рынок (Msty, AnythingLLM, Open WebUI, Cherry Studio) демонстрирует три рабочих паттерна. Для Asterion рекомендуется гибрид Паттерна A + C: Freemium desktop + Pro + Teams/Enterprise governance.

Уровни монетизации

УровеньЧто включаетЦенаДля кого
Community Free Полный open-source core: локальный чат, все AI-модули, Privacy Radar, агенты базовые, RAG, Cookbook $0 навсегда Все, гики, early adopters
Pro Desktop Signed installer + автообновления, E2EE sync между устройствами, premium рецепты, расширенная диагностика, приоритетная поддержка ~$59 one-time или ~$6/мес Power users, профессионалы
Plugin Marketplace Комиссия с продажи MCP-серверов, рецептов, workflow, prompt packs, creative recipes сторонних авторов 20–30% комиссии Экосистема / разработчики
Team Server Multi-user, роли, audit logs, shared workspaces, shared knowledge, policy controls, admin console ~$300/seat/year (мин. 5 seats) Малые команды, стартапы
Enterprise Airgap deployment, white-label, SSO/SAML/AD, SLA, security review, compliance pack (GDPR, HIPAA), custom integrations от $12K–30K/год Корпорации, регулируемые отрасли
Hybrid Cloud «Asterion Credits» — оптовые токены API-провайдеров через единый баланс. Решение для пользователей без GPU. Pay-as-you-go Пользователи без мощного железа

Прогноз и ориентиры рынка

Референс Msty: Freemium free → Pro $149/год → Lifetime $349 → Teams $300/seat/year. AnythingLLM: free self-host → hosted $50–99/мес → enterprise custom. Для Asterion при 5–10K активных пользователей, конверсии 3–7% в Pro/Marketplace + enterprise сделки — устойчивая бизнес-модель.

Ключевое сообщение

«Мы берём деньги только за удобство и масштаб. Ядро и контроль — всегда твои, бесплатно.»

07

MVP и роадмап

MVP (8–12 недель, реально для 1 разработчика)

Цель MVP

Проверить: нужен ли пользователям не просто AI-чат, а рабочая AI-среда с локальным контролем, файлами, моделями, артефактами и прозрачной приватностью.

  • Tauri desktop shell + Ollama интеграция + базовый чат
  • Smart Chat с Model Autopilot и режимами ответа
  • Model Cookbook с hardware scan + Reality Check
  • Privacy Radar (полноценный, перед каждым запросом)
  • Context Rooms + базовый RAG (PDF/TXT → embeddings → Q&A)
  • Базовый Agent Lab: read-only/draft + Task Simulator + Flight Recorder
  • Artifact Panel с экспортом в Markdown/PDF
  • Simple/Pro/Safe режимы + onboarding wizard (5 шагов)

Что не входит в MVP

  • Plugin Marketplace, Team workspaces, Image Studio
  • Browser extension, сложные автоматизации
  • Mobile app, enterprise compliance, encrypted sync

Роадмап по фазам

Phase 1

Phase 3

Phase 4
